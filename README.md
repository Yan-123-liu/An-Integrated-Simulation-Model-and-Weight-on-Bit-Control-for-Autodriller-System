# An-Integrated-Simulation-Model-and-Weight-on-Bit-Control-for-Autodriller-System
This repository contains the MATLAB/Simulink simulation files and source code for the paper "An Integrated Simulation Model and Weight on Bit Control for Autodriller System". The core of this project is a hybrid PSO-Fuzzy-PID-LADRC (Linear Active Disturbance Rejection Control) algorithm designed to precisely control the drawworks system to maintain stable Weight on Bit (WOB). In this simulation, the drawworks system is simplified and represented by a transfer function. The repository includes the main Simulink model (.slx), Particle Swarm Optimization scripts (.m), and the Fuzzy Logic inference system (.fis).
